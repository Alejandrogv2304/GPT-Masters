1. OBJETIVO DEL PROYECTO: ¿Cuál es el objetivo de cada uno de esos proyectos con sus palabras y describa que debe hacer para desarrollarlo?

El objetivo del proyecto es construir una computadora básica utilizando piezas pequeñas como bits, registros y bloques de memoria;  las conectamos entre sí para entender cómo funciona una computadora internamente. Para hacer el proyecto, primero se  crean partes simples que puedan sumar o guardar datos, y luego combinarlas para hacer componentes más grandes como la memoria. Con esto se aprende cómo se construyen los sistemas que hacen que las computadoras sirvan.


2. DIFERENCIA LÓGICA ARITMÉTICA Y LÓGICA SECUENCIAL: Explique las principales diferencias entre la lógica aritmética y la lógica secuencial.

La lógica aritmética hace operaciones matemáticas: suma resta o multiplica, sin guardar información anterior. Los circuitos procesan los datos que entran y generan un resultado basándose únicamente en esas entradas. 

La lógica secuencial guarda la información de lo que ha pasado antes. Los circuitos que usan esta lógica son útiles para cosas como los contadores que necesitan llevar un seguimiento del tiempo o de eventos. No se procesan solo los datos actuales, sino que también se usa la información que ya se guardó antes.

BONUS: ¿Qué tipo de unidades aritmético lógicas existen? 

Podemos llamar a las Unidades aritmético-lógicas como ALU, y hay diferentes tipos de precisión fija, trabajan con números enteros, operaciones simples como suma y resta. Las de punto flotante, más complejas, para manejar decimales. Las de bit a bit se enfocan en operaciones lógicas como AND y OR, manipulando cada bit. Vectoriales, procesan varios datos a la vez, sirve en gráficos y ciencia. Las combinadas mezclan todo lo anterior, se usan en procesadores avanzados para cálculos complicados.
